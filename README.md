# Hướng dẫn tải và cài đặt NeonCyberForce

Trang này hướng dẫn chi tiết cách tải xuống tựa game **NeonCyberForce** từ kho lưu trữ GitHub chính thức tương ứng với từng hệ điều hành và phiên bản cụ thể (ví dụ minh họa với **Phiên bản 6.3**).

---

## 📥 Các bước tải xuống từ GitHub

1. Truy cập trực tiếp vào trang kho lưu trữ (repository) chính thức của dự án tại địa chỉ:  
   👉 [https://github.com/vinh-laptrinhvienVIEa80/Game-NeonCyberForce](https://github.com/vinh-laptrinhvienVIEa80/Game-NeonCyberForce)

2. Điều hướng đến thư mục chứa phiên bản bạn muốn trải nghiệm (ví dụ: thư mục **`NeonCyberForce_v6.3`** tương ứng với Phiên bản lớn 6, Phiên bản nhỏ 3).

3. Lựa chọn và tải về tệp tin phù hợp nhất với hệ điều hành và định dạng máy của bạn theo danh sách dưới đây:

### 🪟 Dành cho hệ điều hành Windows (Hỗ trợ từ v1.0+)
* **Định dạng `.exe`**: Trình cài đặt trực tiếp, nhanh gọn và tự động tạo lối tắt (shortcut) ngoài màn hình.
* **Định dạng `.zip`**: Gói nén dạng portable, chỉ cần giải nén ra là có thể chạy file thực thi bên trong mà không cần cài đặt.

### 🍏 Dành cho hệ điều hành MacOS (Chính thức hỗ trợ từ v4.0+)
* **Định dạng `.app`**: Ứng dụng độc lập chạy trực tiếp trên máy Mac.
* **Định dạng `.dmg`**: Tệp đĩa ảo cài đặt truyền thống, kéo thả vào thư mục Applications cực kỳ mượt mà.
* **Định dạng `.zip`**: Gói nén dự phòng để giải nén thủ công.

### 🐧 Dành cho hệ điều hành Linux (Chính thức hỗ trợ từ v6.0+)
* **Ubuntu & Debian**: Tải file gói cài đặt định dạng **`.deb`** để cài qua trình quản lý gói hoặc lệnh `dpkg`.
* **Red Hat & Fedora**: Tải file gói cài đặt định dạng **`.rpm`** phù hợp với hệ thống nhân Red Hat.
* **Mọi bản phân phối Linux khác**: Tải file nén dạng **`.tar.gz`** để giải nén và chạy linh hoạt trên mọi môi trường.

---

> 💡 **Mẹo nhỏ:** Bạn cũng có thể theo dõi mục **Releases** ở cột bên phải của trang GitHub để tải về các bản build ổn định mới nhất được đóng gói sẵn tự động cho tất cả các nền tảng!
