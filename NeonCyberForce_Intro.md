# NeonCyberForce - Tựa game hành động 2D phong cách GTA bằng Python

**NeonCyberForce** là tựa game hành động 2D thế giới mở lấy cảm hứng từ huyền thoại *Grand Theft Auto (GTA)*, được phát triển hoàn toàn bằng ngôn ngữ lập trình **Python** (sử dụng thư viện Pygame). Trò chơi kết hợp phong cách đồ họa neon cyberpunk rực rỡ, lối chơi tự do và tối ưu hóa mượt mà cho cả các hệ điều hành đời cũ lẫn hiện đại.
